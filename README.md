# StudyCompass - AI Student OS

A unified, AI-powered platform tailored for Indian students planning to study abroad. Designed to act as a definitive "OS" featuring a Copilot mentor, automated SOP/Timeline generators, and an intelligent Loan Funnel.

## Tech Stack
- **Frontend/Backend:** Streamlit
- **Generative AI:** Anthropic Claude (claude-3-5-sonnet-20240620)
- **Data/Visualization:** Pandas, Plotly (for potential extensions)

## Setup Instructions

1. **Clone the repository / open the directory.**
2. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Configure Environment:**
   Copy `.env.example` to `.env` and add your `ANTHROPIC_API_KEY`.
4. **Run the Application:**
   ```bash
   streamlit run app.py
   ```

## Features Complete
- Persistent User Profile with visually engaging Score Card.
- AI Chatbot Mentor with context-awareness.
- Generative Timeline & SOP generator.
- Smart Loan Funnel with Eligibility Scoring and AI insight.
- Proactive Engagement Engine with Next Best Actions.
